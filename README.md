# MyFinControl Backend

Sistema de backend para aplicação de controle financeiro e listas de compras.

## Tecnologias Utilizadas

- Node.js
- TypeScript
- Express.js
- Supabase (PostgreSQL)
- JWT para autenticação
- Bcrypt para hash de senhas

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Testes

```bash
npm test
```

## Lint

```bash
npm run lint
```

## Type Check

```bash
npm run typecheck
```
